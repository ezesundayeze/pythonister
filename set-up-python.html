Windows Python Development Environment Setup

Unlike Linux and MacOS, windows doesn’t ship with Python as of when I wrote this. Maybe they might start doing so in future windows 10 update. But for now there are three ways to install Python on a Windows PC.

1). Downloading from Microsoft Store: 

This is the easiest way to install Python on a Windows PC. You don’t have to worry too much about anything. Just go to the store and search for Python and install it.

Unfortunately, this only works on the newest copy of windows 10.

There are different versions of Python but we are interested in Python3.7. That’s what we’ll be using in this course.

2). Download the installer from Python.org

Follow the steps below to get Python3 from python.org

i). Open your web browser window and navigate to the Download page for Windows at python.org.


ii). below the heading at the top that says Python Releases for Windows, click on the link for the Latest Python 3 Release - Python 3.x.x. (As of this writing, the latest is Python 3.7.3.)

3). Scroll to the bottom and select either Windows x86-64 executable installer for 64-bit or Windows x86 executable installer for 32-bit.

Did you ask why 32-bit and 64-bit disparity?

Here is the thing, for Windows users, you can choose either the 32-bit or 64-bit installer.

The one you should choose depends on some factors:

 If your PC system has a 32-bit architecture, then you should choose the 32-bit installer. Straight up. The 32-bit version will generally use less memory. However, the 64-boit version is not compatible with the 32-bit architecture. So you can’t install 64bit based python installer on a 32-bit machine.

On the other hand. On a 64-bit PC, either python installers will work. Generally, as stated earlier, the 32-bit installer will use less memory but the 64-bit version performs better for applications with intensive computation.

I’ll advise you to install the 64bit version.

iii). Run the Installer

Once you have chosen and downloaded the installer, simply run it by double-clicking on the downloaded file.

Important: You want to be sure to check the box that says Add Python 3.x to PATH as shown to ensure that the interpreter will be placed in your execution path.

Then just click Install Now. That should be all there is to it. A few minutes later you should have a working Python 3 installation on your system.

3). Windows Subsystem for Linux (WSL)

Are you running Windows 10 Creators or Anniversary Update? You have another option to install Python.

These versions of Windows 10 include a feature called the Windows Subsystem for Linux. This allows you to run a Linux environment directly in Windows without the need to install a virtual machine like Virtual Box, or VM ware.

You can find more information about this on  the Microsoft website in the Windows Subsystem for Linux Documentation article.

You can also find instructions on how to enable the subsystem in Windows 10 and install a Linux distribution in this guide(Windows 10 Installation).

Ubuntu

Ubuntu ships with Python, Yey….

Wait…

Run

$ python --version

to know if the version that was shipped is python3.*  if it’s not you might need to run the following commands to install python3.*

$ sudo apt-get update

$ sudo apt-get install python3.7

 

As you can see, installing Python on Ubuntu is a straight-forward process. Run the following commands above and you’re good to go.

MacOS

Installing on MacOS is also pretty simple. If you have brew installed, run

$ brew install python3’

And after a few minutes you are done downloading and installing Python.

And that’s it.
